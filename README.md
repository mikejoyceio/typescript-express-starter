# TypeScript + Express Starter

###### Overview

A barebone TypeScript and Express project starter.

## Getting Started

###### Clone Repo

```
$ git clone https://github.com/mikejoyceio/typescript-express-starter.git
```

###### Install Dependencies

```
$ npm install
```

###### Run Server

```
$ npm run start
```
